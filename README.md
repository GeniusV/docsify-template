# Docsify Template


## Develop

```
npm i docsify-cli -g
npm i docsify-tools -g
```

## Init
```bash
docsify init ./doc
```

## Preview
```bash
docsify serve ./doc
```

## Generate Side Bar

```bash
docsify-auto-sidebar -d docs
```

Add copyright to footer
```html
<footer id="mb-footer" style="margin-left: 15px;margin-top: 10px"></footer>
```
